# 基于之前GNSS深度研究结果的论文写作研究

## 研究总结
## 1. 已覆盖子主题
- **多路径半天球模型/MHM**：已定位到J Geod 2016（10.1007/s00190-015-0870-9）及后续可视化工作（2021），说明MHM用于多路径空间分布建模、展示与改正。
- **城市环境天空图/可见性表示**：已覆盖skyplot、卫星可见性、城市峡谷环境表示。
- **图学习方法**：已获取NAVIGATION 2024图Transformer论文，关注城市峡谷卫星可见性预测。
- **NLOS识别相关基线**：已触达vision-aided、shadow matching、机器学习LOS/NLOS分类等近邻方向。

## 2. 证据强度评估
- **强**：MHM/半天球模型基础文献，来源于J Geod等主流期刊，概念成熟。
- **较强**：图Transformer预测城市卫星可见性，已有明确量化结果：Ctest1准确率**96.29%**，Ctest2 **86.15%**，但环境变化时性能下降。
- **中等**：纯GNSS LOS/NLOS机器学习分类已有零散论文，但当前条目中缺少统一对比与稳定复现证据。
- **弱**：**“纯GNSS + 时空图 + 半天球模型 + NLOS识别”**的直接对应论文尚未找到，更多是相关拼接。

## 3. 尚未覆盖的缺口
- 缺少直接将**MHM/半天球先验**与**时空图神经网络**融合用于**纯GNSS NLOS识别**的核心文献。
- 缺少面向**测量级NLOS判别**而非仅“卫星可见性预测”的工作梳理。
- 缺少统一实验维度：输入特征、标签定义、场景迁移、定位增益。

## 4. 核心发现
- MHM更擅长描述**静态/站点依赖**的多路径空间规律。
- 图Transformer在城市可见性预测上效果突出，但任务更偏**环境遮挡/可见性**，未直接等价于NLOS识别。
- 视觉/地图辅助方法有效，但不符合“**纯GNSS**”约束，因此可作为对照而非最终方案。

## 5. 下一步建议
1) 补检索关键词：`pure GNSS LOS/NLOS classification C/N0 pseudorange residual Doppler graph neural network urban`。
2) 论文写作上将创新点收敛为：**半天球环境先验 + 卫星时空图 + 纯GNSS观测特征**。
3) 建立对比表：MHM、skyplot、visibility prediction、NLOS classification、positioning gain。
4) 若文献仍稀缺，可将题目表述为“**面向纯GNSS城市NLOS识别的半天球先验时空图方法**”，突出方法创新而非文献复现。
